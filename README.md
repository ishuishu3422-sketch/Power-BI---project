# 🛒 Blinket Sales Analysis Dashboard

An interactive Power BI dashboard analyzing sales, outlet performance, and customer ratings for Blinket 

---

##  Project Overview

This dashboard transforms raw sales data into actionable insights for Blinket's business operations. Built as part of my data analytics portfolio, it demonstrates my skills in Power BI, data cleaning, and interactive visualization.

**Dataset:** Excel file with ~8,500 rows and 12 columns
---

##  Business Objectives

- Analyze total sales, average sales, and customer ratings
- Evaluate outlet performance across different city tiers
- Identify top-performing product categories
- Track sales trends by outlet establishment year
---

## 
Key Metrics (KPIs)

| Metric | Description |
|--------|-------------|
| Total Sales | Sum of all revenue |
| Average Sales | Revenue per order |
| Number of Items | Total items sold |
| Average Rating | Customer satisfaction score (3.9/5) |

---

## mc 39Dashboard Features

| Feature | What It Does |
|---------|-------------|
| **Dynamic KPIs** | Real-time updating metrics |
| **Slicers** | Filter by outlet location, size, and item type |
| **Field Parameters** | Toggle between total sales, average sales, items, and ratings |
| **Reset Button** | Clear all filters instantly |
| **Drill-through** | Click charts to explore deeper |

---

##  9Visualizations Included

| Visual | Purpose |
|--------|---------|
| Donut Chart | Sales by fat content |
| Clustered Bar Chart | Sales by item type |
| Line Chart | Sales by outlet establishment year |
| Funnel Chart | Sales by outlet location (Tier 1,2,3) |
| Matrix Table | Detailed breakdown by outlet type |

---

##  Key Insights

| Insight | Finding |
|---------|---------|
|  Top Selling Category | Fruits & Vegetables |
|  Best Performing Location | Tier 3 cities |
|  Highest Sales Outlet Size | Medium outlets |
|  Best Establishment Year | 2018 outlets |
|  Customer Rating | Average 3.9/5 (room for improvement) |
|  Fat Content Trend | Low-fat products dominate sales |

---

##  Tools & Techniques Used

- **Power BI Desktop** — Dashboard creation
- **Power Query Editor** — Data cleaning (fixed "LF" → "Low Fat")
- **DAX** — Custom measures for KPIs
- **Field Parameters** — Dynamic metric switching
- **Slicers & Filters** — Interactivity
- **Custom Branding** — Blinket yellow & green theme
